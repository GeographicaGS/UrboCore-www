How can I check if my connector is bringing data?.

Please check the log file for any errors first.

My connector is running and no errors appear on the log but there is no data entering.

Check your configuration file to see if your connector is using the processing module (processing -> active -> true). If that is the case, change "processing -> active" to false, restart the connector and check the logs again. The errors will appear in the log.

Log shows errors indicating connector is unable to recreate a subscription.

In this case you will need to delete by hand in the database the entries relative to your connector in the table public.subscriptions.
